# Product Alignment Playbook

1. **Evaluation rubric**
   - Align on relevance, hallucination, bias, robustness, safety. Reference `scripts/eval/run_suite.py` output for release gates.
2. **Release checklist**
   - Attach evaluator + governance logs, KPI deltas, and scenario evidence in PRs.
   - Document human-in-the-loop feedback cycles (annotation counts, bias metrics).
3. **Communication cadence**
   - Weekly sync between data, engineering, product to review reward model performance and upcoming experiments.
